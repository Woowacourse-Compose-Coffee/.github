# 컴포즈 커피 스터디
당근 마켓 화면을 Compose로 클론 코딩하면서 선언형 UI와 Android 를 학습합니다.


## 진행방식
### 제출방식
1. 화면 1개당 1주의 시간을 갖습니다.
2. 매주 토요일 13:00까지 PR 제출합니다.
3. 그 이후부터 금주 일요일 23:59까지 리뷰를 완료합니다.

### 리뷰방식
1. 참여인원은 10명
2. 리뷰조는 3, 3, 3명 단위로 3개의 조가 서로를 리뷰한다.
3. 깍두기(나)는 한개 조의 최소 1명을 리뷰한다. 시간나면 더하고ㅋㅋ
4. 리뷰는 평가가 아님. 질문도 좋고 내가 아는거 자랑해도됨.
5. 의미없는 리뷰는 하지말자. ex. LGTM, 좋네요~, 와 그렇군요..
