# 컴포즈 커피 스터디
당근 마켓 화면을 Compose로 클론 코딩하면서 선언형 UI와 Android 를 학습합니다.


## 진행방식
### 제출방법
1. [Compose-Carrot](https://github.com/Woowacourse-Compose-Coffee/Compose-Carrot) 레포지토리를 Fork 받고, (본인 닉네임)-Compose-Carrot 의 이름으로 레포지토리를 만듭니다.
2. 생성한 레포지토리를 로컬로 clone해 가져옵니다.
3. 로컬의 main 브랜치에서 새로운 브랜치로 옮긴 후 작업을 하고, 커밋을 합니다.
4. 새로운 브랜치에 push를 하고, 본인 닉네임으로 된 레포지토리의 main 브랜치에 PR을 작성합니다.

### 제출기한
1. 화면 1개당 1주의 시간을 갖습니다.
2. 매주 토요일 13:00까지 PR 제출합니다.
3. 그 이후부터 금주 일요일 23:59까지 리뷰를 완료합니다.


### 리뷰방식
1. 참여인원은 10명
2. 리뷰조는 3, 3, 3명 단위로 3개의 조가 서로를 리뷰한다.
3. 깍두기(나)는 한개 조의 최소 1명을 리뷰한다. 시간나면 더하고ㅋㅋ
4. 리뷰는 평가가 아님. 질문도 좋고 내가 아는거 자랑해도됨.
5. 의미없는 리뷰는 하지말자. ex. LGTM, 좋네요~, 와 그렇군요..

### 세부 태스크
1. 상품 리스트 화면 구현하기.
   - 마감 3/21(토) 그러나 이미 구현했으면 미리 올리기..
2. 공사중⛏️

### 매주 발표회
1. 해당 주차에 기능을 개발하면서 몰랐던 지식, 깊게 파본 지식, 자랑하고 싶은 코드를 발표합니다.
2. 매주 2명을 선정합니다(지난 주차 발표시 제외).
3. 매주 금(15:00)에 10분내외로 발표합니다.
4. 끝나고 나머지 공부할사람은 남습니다.
